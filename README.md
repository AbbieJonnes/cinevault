# CineVault

A personal movie library app built with React and Tailwind CSS. Users can view, add, and delete movies from their collection.

---

## Built With

- React
- Vite
- Tailwind CSS

---

## Features

- View a list of movies
- Add new movies to the collection
- Delete movies from the list
- Responsive layout with a styled header and footer

---

## Project Structure

```
src/
├── components/
│   ├── Header.jsx
│   ├── Footer.jsx
│   ├── MovieList.jsx
│   └── MovieItem.jsx
└── App.jsx
```

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/AbbieJonnes/cinevault.git
cd cinevault
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
